About the Sustainable Energy Hub Project

This website was created as an educational project to fulfill the requirements of a web design and development course. The subject chosen is **Sustainable Energy**, emphasizing sources like solar, wind, and hydro power.

Project Requirements Demonstrated:

  • **5 Pages:** Home, Solar, Wind, Hydro, and About are all linked.
  • **CSS & JS:** Custom CSS is used for the menu and animation, and a JavaScript function is present.
  • **Anchor:** An internal link (anchor) is used on the Home page.
  • **Design:** Modern CSS (Flexbox) is used for layout instead of tables.

Thank you for reviewing this project.
